☁️ Synchronisation Supabase
1
Créez un compte gratuit sur supabase.com et créez un projet
2
Dans Authentication → Providers, activez Email
3
Dans SQL Editor, exécutez :
create table if not exists books ( id text primary key, user_id uuid references auth.users not null, type text, title text, author text, publisher text, collection text, series text, volume text, year text, genre text, notes text, cover text, added_at bigint, updated_at bigint ); alter table books enable row level security; create policy "Users see own books" on books for all using (auth.uid() = user_id);
Dans Project Settings → API : copiez l'URL et la clé anon public.
